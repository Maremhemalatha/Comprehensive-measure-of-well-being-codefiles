# 🌍 Human Development Index (HDI) Prediction using Machine Learning

## 📌 Project Overview

This project predicts the **Human Development Index (HDI)** of a country using Machine Learning. It is developed using **Python, Flask, HTML, CSS, and Scikit-learn**. The application allows users to enter important socio-economic indicators and predicts the HDI score.

---

## 📖 What is HDI?

The **Human Development Index (HDI)** is a statistical measure developed by the **United Nations Development Programme (UNDP)** to evaluate the overall development of a country.

It is calculated using three key dimensions:

- ❤️ Health (Life Expectancy at Birth)
- 🎓 Education (Expected Years of Schooling and Mean Years of Schooling)
- 💰 Standard of Living (Gross National Income per Capita)

Countries are classified into four categories:

- Very High Human Development
- High Human Development
- Medium Human Development
- Low Human Development
